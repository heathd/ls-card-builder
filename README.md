# Readme - LS Card generator

This is designed to generate card templates which can be printed by [Ivory Graphics](https://www.ivory.co.uk/)

Recommended format:

* Card Size: Trumps
* Material: 350gsm Zanta Games Board
* Cards per pack: 34

The print spec is:

page size

* Width: 68mm 
* Height: 106mm

Final size
* X: 3mm
* Y: 3mm
* W: 62mm
* H: 100mm

Safe area:

* X: 8mm
* Y: 8mm
* W: 52mm
* H: 90mm
